# Student_performance_Analysis_Dashboard

This project demonstrates the development of a comprehensive Student Performance Analysis Dashboard in Power BI. It covers the evaluation of student scores, attendance percentages, study hours, engagement levels, grade distributions, and overall academic results using Power BI visuals and DAX measures.
